# Stripe vs Plaid

## Basics
- **Stripe** → Payment gateway. Handles card payments, wallets, subscription billing.
- **Plaid** → Connects to banks/financial institutions. Used for account verification, ACH payments, balance checks.

## Why Choose
- **Stripe** → For collecting payments online.
- **Plaid** → For securely linking bank accounts and financial data.

## When to Use
- **Stripe** → E-commerce, SaaS billing.
- **Plaid** → Fintech apps needing account access (like Venmo, Robinhood).

## Interview Q&A
**Q: Can Stripe and Plaid be used together?**
Yes. Plaid can verify bank accounts, then Stripe can process ACH payments.
